## 🧾 Description

KeenKeeper is a responsive React-based web application designed to help users maintain and strengthen their friendships. Users can track when they last contacted friends, set communication goals, and log interactions like calls, texts, or video chats.

The app provides a clean UI with dynamic routing, real-time interaction tracking, and visual analytics to better manage relationships.

## ✨ Key Features

### 1. 👫 Friend Management System

View all friends in a responsive grid layout with details like contact status, tags, and last interaction time. Each friend has a dedicated detail page.

### 2. ⚡ Quick Check-In & Timeline Tracking

Users can log interactions (Call, Text, Video) directly from the friend details page. এসব interaction automatically Timeline page-এ add হয় with date & type.

### 3. 📊 Friendship Analytics

A visual Pie Chart shows the distribution of interactions (Call, Text, Video), helping users understand their communication habits.
